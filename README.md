# 🎬 CineFlix

Aplicação web para busca, listagem e recomendação de filmes utilizando a API do The Movie Database (TMDB).

---

## 📌 Sobre o projeto

O **CineFlix** é uma aplicação front-end que consome a API do TMDB para exibir filmes populares, permitir buscas e sugerir filmes aleatórios de forma simples e intuitiva.

---

## 🚀 Funcionalidades

* 🔥 Listagem de filmes populares
* 🔍 Busca por nome de filme
* ❤️ Sistema de favoritos (LocalStorage)
* 📂 Página de favoritos
* 🎲 Botão **"Me Surpreenda"** (filme aleatório)
* ⭐ Avaliação dos filmes em estrelas
* 📅 Exibição da data de lançamento

---

## 🛠️ Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* API do TMDB
* LocalStorage
* SweetAlert2

---

## ▶️ Como executar

1. Baixe ou clone o repositório
2. Abra a pasta do projeto
3. Execute o arquivo `index.html` no navegador
